#  College Helper（大学助手）-后端
### 一.环境配置
直接conda 安装
```
conda env create -f env.yaml
```

```list
django4.2

django-redis4.3.0
py2neo2021.2.3
mysqlclient2.2.0

selenium4.10.0

fasttext
```
